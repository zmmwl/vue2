# 数据资产列表

## 1. 接口描述

* **接口说明** ：获取数据资产列表，**支持根据资产所有者和数据资产名称条件查询，具体分页功能**
* **接口路径** ：asset/GetAssetList
* **请求方式** ：POST

## 2. 输入参数

| **参数名称**   | **类型**    | **是否必传** | **描述**    |
|------------|---------------|----------|---|
| pageNumber | integer       | 否        |分页编码，默认1|
| pageSize   | integer       | 否        |分页大小，每页条数，默认10|
| filters    | array of [Filter] | 否        |过滤参数:<br />描述键值对过滤器，用于条件过滤查询。例如过滤ID、名称、状态等 ``1.若存在多个 `Filter`，`Filter`间的关系为逻辑'与'关系。``<br />2. 若同一个 `Filter`存在多个 `Values`，`Values`间的关系为逻辑或关系。 |

### Filter的key

| 参数名称            | 类型     | 是否必填 | 描述                 |
|-----------------|--------|------|--------------------|
| holderCompany   | string | 否    | 资产所有者              |
| assetName       | string | 否    | 资产名称               |
| dataProductType | String | 否    | 数据资产类型：1数据集 |

## 3. 输出参数

| 参数名称       |类型|描述|
|------------|---|---|
| pagination |Pagination                    | 分页信息   |
| list       |array of [AssetItem]|数据资产列表|

## 5. 数据结构

* **filter数据结构**

| 参数名称   | 参数类型 | 是否必须 | 参数说明 |
|--------| ----------------- | -------- | -------- | 
| key    | string | 否 | 过滤字段 |
| values | array of [String] | 否 | 过滤值 |

### Pagination
| 参数名称   | 参数类型             |参数说明 | 
|--------|------------------| ----- | 
| total   | integer          | 总数量 |     
| pageSize | integer| 分页大小   |      
| pageNumber | integer| 分页页码   |  

### AssetItem 结构

| 参数名称            | 类型     | 描述                 |
|-----------------|--------|--------------------|
| assetId         | string | 主键ID               |
| assetNumber     | string | 数据资产编号             |
| assetName       | string | 数据资产名称             |
| holderCompany   | string | 资产所有者              |
| intro           | string | 资产描述               |
| txId            | string | 交易ID               |
| uploadedAt      | string | 登记时间（也是上链时间）       |
| participantId      | string | 平台ID               |
| entityName  | string | 实体名称               |
| dataProductType | int32  | 数据资产类型：1数据集 |
| enterpriseName  | string | 企业名称               |
