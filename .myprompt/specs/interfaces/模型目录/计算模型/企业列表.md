# 企业列表

## 1. 接口描述

* **接口路径** ：/confidentialComputings/GetEnterpriseList
* **接口说明** ：获取企业列表和模型信息，支持根据模型名字模糊查询
* **请求方式** ：POST

## 2. 输入参数

| 参数名称       | 类型 | 是否必填              | 描述   |
|------------|----|-------------------|------|
| pageSize   | 否  | string            | 每页条数 |
| pageNumber | 否  | string            | 当前页码 |
| filters    | 否  | array of [Filter] | 过滤器  |

## 3. 输出参数

| **参数名称**       | **类型**                | **描述** |
|----------------|-----------------------|--------|
| enterpriseList | array of [Enterprise] | 企业信息   |

## 4. 示例

### （1）输入

```json
{
  "modelTypes": 1
}
```

### （2）输出

```json
{
  "status": "success",
  "code": 200,
  "msg": null,
  "data": {
    "enterpriseList": [
      {
        "platformEnName": "xxx",
        "enterpriseModelList": [
          {
            "modelDetailId": "1",
            "modelDetailName": "xxx"
          }
        ]
      }
    ]
  }
}
```

## 5. 数据结构

### Filter

| 参数名称      | 类型     | 是否必填 | 描述   |
|-----------|--------|------|------|
| modelName | string | 否    | 模型名称 |

### Enterprise

| 参数名称                      | 类型                               | 描述   |
|---------------------------|----------------------------------|------|
| participantId            | string                           | 平台ID |
| enterpriseModelDetailList | array of [EnterpriseModelDetail] | 模型列表 |

### EnterpriseModelDetail

| 参数名称            | 类型     | 描述   |
|-----------------|--------|------|
| modelDetailId   | string | 模型ID |
| modelDetailName | string | 模型名称 |
