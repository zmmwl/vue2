# 获取全网计算模型

## 1. 接口描述

* **接口名称** ：/confidentialComputings/model/globalGetAll
* **接口说明** ：获取全网计算模型
* 请求方式：POST

## 2. 输入参数

| **参数名称** | **必选** | **类型**  | **描述** |
| ------------------ | -------------- | --------------- | -------------- |
| pageSize       | 是  | String | 每页条数 |
| pageNumber     | 是  | String | 当前页码 |
| filters        | 是  | Array of [Filter] | 过滤器   |

### Filter的key

| 参数名称          | 类型  |     是否必填    | 描述         |
|---------------|------|--------|------------|
| status | string | 否 | 计算模型目录上链状态 |
| name   | string | 否 | 计算模型名称     |
| entityName   | string | 否 | 企业名称       |
| methodName |  String      | 否  | 方法名称       |


### Filter的value

| 参数名称          | 类型                | 是否必填 | 描述      |
|---------------|-------------------|------|---------|
| status的内容     | Array of [String] | 否    | 计算模型目录上链状态 |
| entityName的内容 | Array of [String] | 否    | 企业名称   |
| name的内容     | Array of [String] | 否    | 计算模型名称 |
| methodName的内容 | Array of [String] | 否    | 方法名称   |


## 3. 输出参数

| **参数名称** | **类型** | **描述** |
| -----------------| -------| --------------------------------------------------------------------- |
| pagination |Pagination                    | 分页信息   |
| list       |array of [AssetItem]|数据资产列表|

## 5. 数据结构

* **filter数据结构**

| 参数名称   | 参数类型 | 是否必须 | 参数说明 |
|--------| ----------------- | -------- | -------- | 
| key    | string | 否 | 过滤字段 |
| values | array of [String] | 否 | 过滤值 |

### Pagination
| 参数名称   | 参数类型             |参数说明 | 
|--------|------------------| ----- | 
| total   | integer          | 总数量 |     
| pageSize | integer| 分页大小   |      
| pageNumber | integer| 分页页码   |  

### list 结构

| 参数名称              | 类型                                                  | 描述                                 |
|-------------------|-----------------------------------------------------|------------------------------------|
| modelId           | String                                              | 计算模型id                           |
| participantId     | String                                              | 组织id                               |
| entityName        | String                                                | 组织名称                               |
| chainId           | uint32                                                | 链id                                |
| chainName         | String                                              | 链名称                                |
| name              | String                                              | 计算模型名称                           |
| type              | String                                              | 计算模型类型                           |
| version           | String                                              | 计算模型版本                           |
| category          | String                                              | 计算模型类别                           |
| description       | String                                              | 计算模型描述                           |
| methodName        | String                                              | 方法名称                               |
| methodDescription | String                                              | 方法描述                               |
| modelFile         | array of [byte]                                     | 计算模型文件                           |
| modelFileHash     | String                                              | 计算模型模型文件哈希                       |
| sourceFile        | array of [byte]                                     | 计算模型数据源文件                        |
| sourceFileHash    | String                                              | 计算模型数据源文件哈希                      |
| onChainStatus     | int32                                               | 上链状态，取值： 100-上链成功,101-上链中,102-上链超时 |
| modelParameters   | array of [ConfidentialComputingModelParameters]     | 计算模型参数                           |
| returnParameters  | array of [ConfidentialComputingModelReturnParameters] | 计算模型返回参数                         |
| createTime        | String                                              | 创建时间                               |
| programmingLanguage | String                                              | 编程语言                               |
| status            | int32                                               | 上架状态    1已上架  2已下架     0空状态        |

### modelParameters 结构

| 参数名称   | 参数类型             |参数说明 | 
|--------|------------------| ----- | 
| fid   | String | 计算模型参数id |     
| name  | String | 计算模型变量名 |      
| dataSource | String | 计算模型数据来源   |  
| dataType   | int    | 计算模型数据类型   |     
| isEncrypt  | int    | 计算模型是否加密   |      
| description| String | 计算模型参数说明   | 


### returnParameters 结构

| 参数名称          | 类型               | 描述                   |
|---------------|------------------|----------------------|
| fid          | String  | 计算模型返回参数id      |
| name         | String  | 计算模型返回参数名称     |
| dataType     | int     | 计算模型返回参数数据类型 |
| isRequired   | int     | 计算模型返回参数是否必填 |
| description  | String  | 计算模型返回参数说明     |
