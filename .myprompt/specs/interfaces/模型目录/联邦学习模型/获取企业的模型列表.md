# 获取企业的联邦学习模型列表

## 1. 接口描述

* **接口名称** ：federatedAI/model/getEnterpriseList
* **接口说明** ：获取企业的联邦学习模型列表
* 请求方式：POST

## 2. 输入参数

| **参数名称**  | **必选** | **类型** | **描述** |
|-----------|--------|--------|--------|
| modelName | 否      | String | 模型名称   |


## 3. 输出参数

| **参数名称**       | **类型**                | **描述** |
|----------------|-----------------------|--------|
| enterpriseList | array of [Enterprise] | 企业信息   |

## 4. 示例



### （2）输出

```json
{
  "code": 200,
  "msg": "",
  "data": {
    "enterpriseList": [
      {
        "platformEnName": "idaoneEn",
        "enterpriseModelList": [
          {
            "modelId": "uuid",
            "federatedId": "202502260324190733500"
          },
          {
            "modelId": "uuid",
            "federatedId": "202502260324190733500"
          }
        ]
      }
    ]
  }
}

```

### Enterprise

| 参数名称                | 类型                         | 描述     |
|---------------------|----------------------------|--------|
| platformEnName      | string                     | 平台英文名  |
| enterpriseModelList | array of [EnterpriseModel] | 企业模型列表 |

### EnterpriseModel

| 参数名称            | 类型     | 描述       |
|-----------------|--------|----------|
| modelId         | string | 模型ID     |
| federatedId     | string | 联邦学习模型id |
