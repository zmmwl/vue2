# 联邦学习模型详情

## 1. 接口描述

* **接口名称** ：/federatedAI/model/getModelDetail
* **接口说明** ：获取联邦学习模型详情
* 请求方式：POST

## 2. 输入参数

| **参数名称** | **必选** | **类型** | **描述** |
|----------|--------|--------|--------|
| modelId  | 是      | String | 模型ID   |

## 3. 输出参数

| 参数名称           | 参数类型   | 参数说明                              |
|----------------|--------|-----------------------------------|
| status         | int    | 0 未发布 1 已发布                       |
| createTime     | string | 创建时间                              |
| chainInfoId    | int    | 链Id                               |
| txId           | string | 交易Id                              |
| onChainStatus  | int    | 上链状态，取值：100-上链成功,101-上链中,102-上链超时 |
| onChainTime    | string | 上链时间                              |
| chainName      | string | 链名称                               |
| modelId        | string | 模型Id                              |
| federatedId    | string | 联邦学习模型id                          |
| modelName      | string | 模型名称                              |
| metrics        | string | 评估结果                              |
| updateTime     | string | 修改时间                              |
| creatorName    | string | 创建人                               |
| description    | string | 描述                                |
| updaterName    | string | 最后修改人                             |
| enterpriseName | string | 企业名称                              |
| partyName      | string | 平台名称                              |
| partyId        | string | 平台id                              |
