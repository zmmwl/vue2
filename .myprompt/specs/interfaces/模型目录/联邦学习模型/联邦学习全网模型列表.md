# 获取全网联邦学习模型列表

## 1. 接口描述

* **接口名称** ：/federatedAI/model/getGlobalList
* **接口说明** ：获取全网联邦学习模型列表
* 请求方式：POST

## 2. 输入参数

| **参数名称**   | **必选** | **类型**            | **描述** |
|------------|--------|-------------------|--------|
| pageSize   | 是      | String            | 每页条数   |
| pageNumber | 是      | String            | 当前页码   |
| filters    | 是      | Array of [Filter] | 过滤器    |

### Filter的key

| 参数名称           | 类型     | 是否必填 | 描述  |
|----------------|--------|------|-----|
| modelName      | String | 否    | 字符串 |
| enterpriseName | String | 否    | 企业名称 |




### Filter的value

| 参数名称     | 类型                | 是否必填 | 描述   |
|----------|-------------------|------|------|
| modelName的内容 | Array of [String] | 否    | 模型名称 |



## 3. 输出参数

| **参数名称**   | **类型**                    | **描述** |
|------------|---------------------------|--------|
| pagination | Pagination                | 分页信息   |
| list       | array of [federatedModel] | 算力分组信息 |

## 5. 数据结构

### filter数据结构

| 参数名称   | 参数类型              | 是否必须 | 参数说明 | 
|--------|-------------------|------|------| 
| key    | string            | 否    | 过滤字段 |     
| values | array of [String] | 否    | 过滤值  |      

### Pagination

| 参数名称       | 参数类型    | 参数说明 | 
|------------|---------|------| 
| total      | integer | 总数量  |     
| pageSize   | integer | 分页大小 |      
| pageNumber | integer | 分页页码 |
