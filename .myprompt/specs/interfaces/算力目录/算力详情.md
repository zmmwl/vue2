# 算力详情

## 1. 接口描述

* **接口名称** ：/computingResource/getGlobalGroupDetailList
* **接口说明** ：获取算力分组详情
* 请求方式：POST

## 2. 输入参数

| **参数名称** | **必选** | **类型**  | **描述** |
| ------------------ | -------------- | --------------- | -------------- |
| groupID           | 是             | String          | 分组ID   |


## 3. 输出参数

| **参数名称** | **类型** | **描述**                            |
| ------------------ | -------------- |-----------------------------------|
| participantId       | String  | 组织ID                              |
| entityName       | String  | 组织名称                              |
| groupID       | String  | 分组ID                              |
| groupName     | String  | 分组名称                              |
| transactionID | String  | 链上交易ID                            |
| description   | String  | 分组描述                              |
| status        | String  | 上链状态 1-上链，-1-未上链                  |
| createTime    | String  | 创建时间                              |
| updateTime    | String  | 修改时间                              |
| chainInfoId   | int     | 链Id                               |
| txId          | String  | 交易Id                              |
| onChainStatus | Integer | 上链状态，取值：100-上链成功,101-上链中,102-上链超时 |
| onChainTime   | String  | 上链时间                              |
| chainId       | String  | 链Id                               |
| chainName     | String  | 链名称                               |
| computingResourceNodeVOs | Array of [ComputingResourceNode] | 节点信息列表                            |
| computingResourceCardVOs | Array of [ComputingResourceCard] | 板卡信息列表                            |

## 5. 数据结构

### computingResourceNodeVOs

| 参数名称     | 参数类型 | 参数说明   |
| ----------- | -------- | ----------|
| nodeID            | String | 节点ID  |
| groupID           | String | 分组ID  |
| nodeName          | String | 节点名称 |
| nodeAddress       | String | 节点地址 |
| nodeSpec          | String | 节点规格 |
| status            | int    | 0-可用，1-不可用，2-已删除 |
| createTime        | String | 创建时间 |
| updateTime        | String | 修改时间 |
| computingResourceCardVOs | Array of [computingResourceCard] | 板卡信息 |


### computingResourceCardVOs

| 参数名称     | 参数类型 | 参数说明  |
| ----------- | -------- | ---------|
| nodeID      | String | 节点ID      |
| cardIndex   | String | 板卡列表序号 |
| cardSerial  | String | 板卡系列号 |
| cardModel   | String | 板卡型号   |
| cardSpec    | String | 板卡配置   |
| cardVersion | String | 板卡版本   |
