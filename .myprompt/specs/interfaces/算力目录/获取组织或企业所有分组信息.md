# 算力详情

## 1. 接口描述

* **接口名称** ：/computingResource/getGlobalGroupInfo
* **接口说明** ：根据partyId获取全网算力分组信息
* 请求方式：POST

## 2. 输入参数

| **参数名称**      | **必选** | **类型** | **描述**                          |
|---------------|--------|--------|---------------------------------|
| participantId | 否      | String | 组织ID                            |
| groupType     | 否      | int32  | 组类别  不传为0 返回全部 1 板卡组信息 |
| entityName    | 否  | string | 企业名称，不传返回全部  |


## 3. 输出参数

| **参数名称** | **类型** | **描述**                                                        |
| ------------------ | -------------- | --------------------------------------------------------------------- |
| computingResourceGroupList | Array of [ComputingResourceGroup] | 分组信息列表|

## 5. 数据结构

### computingResourceGroup

| 参数名称          | 参数类型    | 参数说明   |
|---------------|---------|-----------------------------------|
| participantId | String  | 组织ID |
| groupID       | String  | 分组ID |
| groupName     | String  | 分组名称 |
| transactionID | String  | 链上交易ID |
| description   | String  | 分组描述   |
| status        | String  | 上链状态 1-上链，-1-未上链 |
| createTime    | String  | 创建时间 |
| updateTime    | String  | 修改时间 |
| chainInfoId   | int     | 链Id    |
| txId          | String  | 交易Id  |
| onChainStatus | Integer | 上链状态，取值：100-上链成功,101-上链中,102-上链超时 |
| onChainTime   | String  | 上链时间 |
| chainId       | String  | 链Id |
| chainName     | String  | 链名称 |
| entityName    | String  | 企业名称 |
