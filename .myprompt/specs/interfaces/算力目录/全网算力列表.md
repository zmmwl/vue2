# 获取“全网”分组算力列表

## 1. 接口描述

* **接口名称** ：/computingResource/getGlobalGroupList
* **接口说明** ：显示全网算力分组
* 请求方式：POST

## 2. 输入参数

| **参数名称**   | **必选** | **类型**            | **描述** |
|------------|--------|-------------------|--------|
| pageSize   | 是      | String            | 每页条数   |
| pageNumber | 是      | String            | 当前页码   |
| filters    | 是      | Array of [Filter] | 过滤器    |

### Filter的key

| 参数名称       | 类型     | 是否必填 | 描述  |
|------------|--------|------|-----|
| groupName  | String | 否    | 字符串 |
| status     | String | 否    | 字符串 |
| groupType  | String | 否 | 字符串 |
| entityName | String | 否 | 字符串 |



### Filter的value

| 参数名称          | 类型                | 是否必填 | 描述      |
|---------------|-------------------|------|---------|
| groupName的内容  | Array of [String] | 否    | 组织ID    |
| status的内容     | Array of [String] | 否    | 当前上下架状态 |
| groupType的内容  | Array of [String] | 否    | 分组类型    |
| entityName的内容 | Array of [String] | 否    | 所有者名称   |


## 3. 输出参数

| **参数名称**   | **类型**                                | **描述** |
|------------|---------------------------------------|--------|
| pagination | Pagination                            | 分页信息   |
| list       | array of [computingResourceGroup] | 算力分组信息 |

## 5. 数据结构

### filter数据结构

| 参数名称   | 参数类型              | 是否必须 | 参数说明 | 
|--------|-------------------|------|------| 
| key    | string            | 否    | 过滤字段 |     
| values | array of [String] | 否    | 过滤值  |      

### Pagination

| 参数名称       | 参数类型    | 参数说明 | 
|------------|---------|------| 
| total      | integer | 总数量  |     
| pageSize   | integer | 分页大小 |      
| pageNumber | integer | 分页页码 | 

### computingResourceGroupInfos

| 参数名称          | 参数类型    | 参数说明                              |
|---------------|---------|-----------------------------------|
| participantId | String  | 组织ID                              |
| entityName    | String  | 实体名称                              |
| groupID       | String  | 分组ID                              |
| groupName     | String  | 分组名称                              |
| transactionID | String  | 链上交易ID                            |
| description   | String  | 分组描述                              |
| status        | String  | 上链状态 1-上链，-1-未上链                  |
| createTime    | String  | 创建时间                              |
| updateTime    | String  | 修改时间                              |
| chainInfoId   | int     | 链Id                               |
| txId          | String  | 交易Id                              |
| onChainStatus | Integer | 上链状态，取值：100-上链成功,101-上链中,102-上链超时 |
| onChainTime   | String  | 上链时间                              |
| chainId       | uint32  | 链Id                               |
| chainName     | String  | 链名称                               |
| groupType     | int32   | 算力类型 0 板卡算力         |
