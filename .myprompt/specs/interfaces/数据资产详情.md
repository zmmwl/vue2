# 数据资产详情

## 1. 接口描述

* **接口说明** ：根据数据资产id，获取数据资产详情
* **接口路径** ：asset/GetAsset
* **请求方式** ：POST

## 2. 输入参数

| 参数名称                    | 类型     | 是否必填 | 描述              |
|-------------------------|--------|------|-----------------|
| assetId                 | string | 否    | 数据资产id          |
| assetEnName             | string | 否    | 数据资产英文名字        |
| assetOwnerParticipantId | string | 否    | 数据资产的所属方的参与方的id |

assetId 和 (assetEnName + assetOwnerParticipantId) 不能同时为空
## 3. 输出参数

| **参数名称**  | **类型**    | **描述** |
|-----------|-----------|--------|
| assetInfo | AssetInfo | 响应数据   |

## 5. 数据结构

### AssetInfo 结构

| 参数名称            | 类型       | 描述                                      |
|-----------------|----------|-----------------------------------------|
| assetId         | string   | 主键ID                                    |
| assetNumber     | string   | 数据产品编号                                  |
| holderCompany   | string   | 数据产品所有者                                 |
| assetName       | string   | 数据产品名称                                  |
| assetEnName     | string   | 资产英文简称                                  |
| assetType       | integer  | 数据类型，1-个人数据，2-企业数据，3-公共数据               |
| scale           | string   | 数据规模                                    |
| cycle           | string   | 更新周期                                    |
| timeSpan        | string   | 时间跨度                                    |
| txId            | string   | 交易id                                    |
| intro           | string   | 数据产品简介                                  |
| dataInfo        | DataInfo | 数据集信息                                   |
| participantId   | string   | 平台ID                                    |
| entityName      | string   | 实体名称                                    |
| dataProductType | int32    | 数据资产类型：1数据集，                      |
| scaleUnit       | int32    | // 数据规模单位，1-MB,2-GB,3-TB,4-PB,5-EB,6-ZB |

### DataInfo

| 参数名称      | 类型                             | 描述    |
|-----------|--------------------------------|-------|
| dbName    | string                         | 数据库名称 |
| tableName | string                         | 数据表名称 |
| itemList  | array of [SaveTableColumnItem] | 字段信息  |

### SaveTableColumnItem 结构

| 参数名称         | 类型      | 描述                |
|--------------|---------|-------------------|
| name         | string  | 字段名称              |
| dataType     | string  | 字段类型              |
| dataLength   | string  | 字段长度              |
| description  | string  | 字段描述              |
| isPrimaryKey | integer | 是否为主键：1.否，2.是     |
| privacyQuery | integer | 是否隐私查询：1是，0否。 默认1 |
